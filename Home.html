Comandos Grid

Lista de comandos

1. Display

Define un contenedor flexible, permitiendo alinear sus hijos directos de la forma que determinemos. .container { display:flex / inline-flex; }

2.Flex-direction / flex-wrap

Direccion de los item en fila o columna. .container{ flex-direction: row / row-reverse / column / column-reverse; } .container{ flex-wrap: nowrap / wrap / wrap-reverse; }

3.Flex-flow (flex-direction + flex-wrap)

.container{ flex-flow: row (flex-direction) wrap (flex-wrap); }

4.Justify-content / Align-item

Alinea los item en sentido horizontal y vertical. .container { /* Alineacion horizontal de los items */ justify-content: flex-start / flex-end / center / space-between / space-around / space-evenly } .container{ /*Alineacion vertical de los items align-items: flex-start / flex-end / center / stretch / baseline }

5. Align-content

Alinea los items en sentido vertical y horizontal .container{ align-content:flex-start / flex-end / center / space-between / space-around / stretch; }

6.flex-grow / flex-shink / flex-basis

.item:nth-child(4) { /*Define en que proporcion crece un item */ flex-grown: 3 (numero); /* por defecto es 0 */ } .item{ /*Define la capacidad de un objeto flexible para contraerse si es necesario */ flex-shrink: 2 (numero); /*por defecto es 1 */ /*Define la capacidad de un objeto flexible para contraerse si es necesario */ flex-basis: auto / 10% / 5rem; }

7. Order

.item { /*Orden de los items, permitiendo colocar un elemento en un orden diferente al que le corresponde en el codigo */ order: 5 (numero); /* por defecto es 0 */ }

8.align-self

Permite alinear de forma individual un elemento .item:nth-child (3) { align-self: auto / flex-start / flex-end / center / baseline / stretch; }
